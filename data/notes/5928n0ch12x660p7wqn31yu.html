Install Fonts

https://itsfoss.com/install-fonts-ubuntu/

User space

  1. Create .fonts in the home directory
  2. Move font files in to the ~/.fonts folder
  3. Reload the fonts cache sudo fc-cache -f -v

Share space

  1. Create a folder in /usr/local/share/fonts
  2. Move font files into that folder

Recommend fonts

  • Nerd fonts: https://www.nerdfonts.com/font-downloads
