Dynamodb

DynamoDB - Amazon DynamoDB

Intro

  • No SQL
  • High performance & scalable
  • AWS service

Features

  • Supports key-value & document data models
  • Low latency
    • in-mem cache (DAX - DynamoDB accelerator)
  • Automated global replication
  • Serverless: automatically scales table
    • maintains performance with zero administration
  • ACID server-side transactions
  • Encryption at rest
    • ecrypts all customer data by default
    • using the AWS Key Management Services
