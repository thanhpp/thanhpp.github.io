
# i3 is a window tilting manager

## Install i3

- [homepage](https://i3wm.org/downloads/)
- Install through apt
- remember to add the gnome-keyring.deb

## Config

- placed in `$HOME/.config/i3` (can be a soft link)
- scripts
    - i3exit: lock, suspend, shutdown
- backgrounds
- command `ln -s /home/thanhpp/go/src/github.com/thanhpp/Ubuntu_Settings/settings/i3 /home/thanhpp/.config/i3`   

## Uninstall

`sudo apt-get remove i3; sudo apt-get purge i3; sudo apt-get autoremove; sudo apt-get autoclean`

`sudo apt remove i3-wm and i3-wm-dbg`
