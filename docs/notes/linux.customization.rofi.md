# Rofi: A window switcher, application launcher and dmenu replacement

> https://github.com/davatorium/rofi

## Install 

```
apt install rofi
```

## Config

- placed in `mkdir -p ~/.config/rofi`
- soft link
- dump current config `rofi -dump-config > ~/.config/rofi/config.rasi`
